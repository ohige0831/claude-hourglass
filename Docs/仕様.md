# Claude Hourglass 仕様

## 概要

Claude Hourglass は、Claude Code の statusLine から取得できる使用制限情報を記録し、Windows のタスクトレイから現在の使用状況を砂時計風に確認できる常駐アプリである。

普段はトレイアイコンとして常駐し、ホバーまたはクリックで現在の 5時間制限・7日制限の使用率とリセット時刻を表示する。アプリ画面を開くと、日別・週別・セッション別の使用傾向をグラフで確認できる。

## 目的

- Claude Code の使用制限を一目で把握する
- リセットまでの残り時間をデスクトップ上で確認する
- 後から日別・週別の利用傾向を見返せるようにする
- TUI操作やWeb画面OCRに依存せず、statusLine のJSONを利用して安定して記録する

## MVP

- Claude Code statusLine から rate_limits 情報を取得する
- `usage.sqlite` にスナップショットを保存する
- Windows タスクトレイに常駐する
- ホバーで現在の使用率・残り時間・リセット時刻を表示する
- クリックで小型の砂時計表示を開く
- アプリ画面で日別・週別の簡易グラフを表示する

## 取得データ

- `captured_at`
- `session_id`
- `model.display_name`
- `rate_limits.five_hour.used_percentage`
- `rate_limits.five_hour.resets_at`
- `rate_limits.seven_day.used_percentage`
- `rate_limits.seven_day.resets_at`
- `cost.total_cost_usd`
- `context_window.current_usage`
- `version`

## 注意点

`rate_limits.*.used_percentage` は現在の制限枠に対する使用率であり、厳密な日別使用量そのものではない。

日別・週別の利用量グラフは、スナップショット差分、セッションID、cost、token情報を組み合わせて推定する。
